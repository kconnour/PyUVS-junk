Ancillary files#

Once you’ve downloaded the data from the OneDrive link from the installation, you can easily access the ancillary files with these functions.

Maps#

pyuvs.load_map_magnetic_field_closed_probability()

Load the map denoting the probability of a closed magnetic field line.

pyuvs.load_map_magnetic_field_open_probability()

Load the map denoting the probability of an open magnetic field line.

pyuvs.load_map_mars_surface()

Load the Mars surface map.

Templates#

pyuvs.load_template_wavelengths()

Load the wavelength centers that correspond to the MUV templates.

pyuvs.load_template_co_cameron()

Load the normalized MUV CO Cameron bands template.

pyuvs.load_template_co_plus_1st_negative()

Load the normalized MUV CO + 1NG (first negative) bands template.

pyuvs.load_template_co2_plus_fdb()

Load the MUV CO 2 + FDB (Fox-Duffendack-Barker) bands template.

pyuvs.load_template_co2_plus_uvd()

Load the MUV CO 2 + UVD (ultraviolet doublet) template.

pyuvs.load_template_n2_vk()

Load the MUV N 2 VK (Vegard-Kaplan) bands template.

pyuvs.load_template_no_nightglow()

Load the MUV NO nightglow bands template.

pyuvs.load_template_oxygen_2972()

Load the MUV oxygen 297.2 nm template.

pyuvs.load_template_solar_continuum()

Load the MUV solar continuum template.

Instrument#

Generic#

pyuvs.load_voltage_correction_voltage()

Load the voltages used in the voltage correction.

pyuvs.load_voltage_correction_coefficients()

Load the voltages fit coefficients used in the voltage correction.

Far-ultraviolet#

pyuvs.load_fuv_sensitivity_curve_manufacturer()

Load the manufacturer's FUV detector factory sensitivity curve.

Mid-ultraviolet#

pyuvs.load_muv_flatfield()

Load the mid-hi-resolution flatfield created from data taken during the Mars year 34 global dust storm.

pyuvs.load_muv_point_spread_function()

Load the MUV point spread function.

pyuvs.load_muv_sensitivity_curve_manufacturer()

Load the manufacturer's MUV detector factory sensitivity curve.

pyuvs.load_muv_sensitivity_curve_observational()

Load the MUV detector sensitivity curve derived from observations.
